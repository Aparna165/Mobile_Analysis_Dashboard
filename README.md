📊 Mobile Sales Dashboard (Excel Project)
📌 Project Overview

This project presents a Mobile Sales Dashboard created using Microsoft Excel. It helps analyze sales performance, product trends, and regional distribution through interactive visuals. The dashboard is designed to make data easy to understand and support better business decisions.
<img width="843" height="625" alt="Dashboard" src="https://github.com/user-attachments/assets/5ef123cf-fb06-4e15-9df9-08218a93b65a" />

🎯 Objectives
Analyze mobile sales performance across regions
Identify top-performing products and brands
Track monthly and yearly sales trends
Provide insights using KPIs and visual charts
📂 Dataset Details

The dataset includes the following key fields:

📅 Date (Year, Month)
🌍 Region / Country
📱 Product / Brand
💰 Sales Amount
🔢 Quantity Sold
📊 Dashboard Features
✔️ Interactive Pivot Tables
✔️ Dynamic Charts (Bar, Line, Pie)
✔️ KPI Cards for quick insights
✔️ Filters (Slicers) for easy data exploration
✔️ Clean and user-friendly layout
📈 Key KPIs
Total Sales Revenue
Total Units Sold
Top Performing Brand
Best Sales Month
Highest Revenue Region
🔍 Insights Generated
Identified top-selling mobile brands
Found peak sales months and trends
Compared regional performance
Highlighted underperforming products
🛠️ Tools & Techniques Used
Microsoft Excel
Pivot Tables
Pivot Charts

Data Cleaning & Formatting

Download the Excel file
