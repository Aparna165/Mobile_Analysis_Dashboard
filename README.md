📊 Mobile Sales Analysis Dashboard (Excel Project)

📌 Project Overview

This project presents a Mobile Sales Dashboard created using Microsoft Excel. It helps analyze sales performance, product trends, and regional distribution through interactive visuals. The dashboard is designed to make data easy to understand and support better business decisions.

🎯 Objectives
- Analyze mobile sales performance across different countries
- Identify top-performing regions and products
- Track monthly and yearly sales trends
- Build an interactive dashboard for better decision-making

📂 Dataset Description
The dataset includes:
- 🌍 Country-wise sales data
- 📅 Monthly and yearly records
- 📱 Product/Brand details
- 💰 Price/Sales values

📊 Dashboard Features

🔹 KPI Cards
- Total Sales
- Total Orders
- Number of Countries
- Average Sales Value

🔹 Charts Used
- 📌 Bar Chart → Country-wise Sales
- 📌 Line Chart → Monthly Trends
- 📌 Pie Chart → Brand Distribution
- 📌 Column Chart → Year-wise Comparison

🔹 Filters / Slicers
- Year
- Month
- Country
- Brand

🔍 Key Insights
- Certain countries contribute significantly to overall revenue
- Sales show variation across months, indicating seasonal trends
- Some brands dominate the market compared to others
- Growth patterns can be observed across different time periods

⚠️ Problem Statements & Solutions

1. Uneven Sales Distribution
- Problem: Few countries dominate sales
- Solution: Expand marketing in low-performing regions

2. Seasonal Fluctuations
- Problem: Sales vary across months
- Solution: Plan promotions during low-sales periods

3. Brand Performance Gap
- Problem: Some brands underperform
- Solution: Improve product positioning and pricing strategy

4. Limited Customer Reach
- Problem: Sales concentrated in specific regions
- Solution: Target new markets

5. Data Visibility Issues
- Problem: Raw data is hard to interpret
- Solution: Dashboard simplifies insights visually

🛠 Tools & Technologies

- Data Cleaning Techniques
